== So you want to start a blog, eh?

Before you start, typing relentlessly on your keyboard and broadcast your ideas to the world, here's a few things that you may want to check out.

How's the editor?

Glad you asked! This blog engine has a few assumptions that I'd like to share with you. First and foremost, this is built by a web developer for developers. This means that there's the editor provides a little bit more flexibility than other blogging platform you may know. Features such as:

  • HTML editor
  • CSS editor
  • Javascript editor
  • Partials editor and importer
  • Image importer

Can give you the freedom to build each and every post the way you intended them to be. I'm pretty sure a wall of text isn't really telling so here's some visuals to help you understand what this all means!
